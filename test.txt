# Redis CLI 사용하는 법

## 개요
Redis CLI를 사용하여 Redis 서버에 접속하고 관리하는 방법에 대한 가이드입니다.

## 설치
설치는 블로그 참조.

## 사용법

### 기본 연결 명령어
Redis CLI로 원격 서버에 확인하려면 아래 명령어를 사용합니다:

```shell
# redis-cli로 확인하려면 아래 명령어 사용
redis-cli -h <호스트주소>
```
